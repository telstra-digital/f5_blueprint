Delete a Virtual
DELETE
/mgmt/tm/ltm/virtual/{name}{?ver,expandSubcollections}

Delete a Virtual. The current user must be the same user who created the Virtual. It returns the deleted Virtual on success.

  • Parameters
  • name
    string (required) Example: test_vs

    The name of the virtual server.

    ver
    string (optional) Example: 11.5.0

    The version of the API to consume.

    expandSubcollections
    string (optional) Example: true

    Expand the subcollection of the API.

  • Response  200
